Header tag

Header Tag is used to display heading part of the webpage. usually company logo, name are displayed in this section.
nav tag (short of navigation ) is used to add navigation between different pages

Section Tag

The section tag defines the section between the document.

The article tag specifies independent, self-contained content. An article should make sense on its own and it should be possible to distribute it independently from the rest of the site. Potential sources for the article element: Forum post Blog post News story

Aside Tag

Aside tag contains some other information about the main content which are indirectly related

Footer tag

The footer tag defines a footer for a document or section. A footer element typically contains:

  • authorship information
  • copyright information
  • contact information
  • sitemap
  • back to top links
  • related documents
You can have several footer elements in one document.
